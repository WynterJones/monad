Getting Started

1

Installation

npm create monad@latest my-site
cd my-site
npm install
npm run dev
2

Project Structure

my-site/
├── pages/          # Your HTML pages
├── partials/       # Reusable components
├── public/         # Static assets
├── src/            # JS/CSS source
├── collections.json # Data for loops
└── vite.config.ts  # Monad config

Feature Configuration

Performance Optimization

// vite.config.ts
monad({
  performance: {
    enabled: true,
    preloadCriticalCss: true,
    addFetchPriority: true,
    lazyLoadImages: true
  }
})

Image Optimization

images: {
  enabled: true,
  formats: ["webp", "avif"],
  responsive: {
    enabled: true,
    breakpoints: [640, 768, 1024, 1280]
  }
}

SEO & Auditing

audit: {
  enabled: true,
  mode: "error" // or "warn"
},
sitemap: { enabled: true },
robots: { enabled: true }

Best Practices

Use Semantic HTML

Monad respects your markup. Use proper heading hierarchy and semantic elements.

Optimize Images

Let Monad handle image optimization. Use high-quality sources and appropriate formats.

Monitor Build Reports

Check dist/__monad/report.html for SEO and performance insights.

Reuse Partials

Create partials for repeated elements to maintain consistency.
