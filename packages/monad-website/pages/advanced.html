Advanced Collections

Use index tracking and complex data in your loops.

// collections.json
{
  "products": [
    { "name": "Pro", "price": 99, "featured": true },
    { "name": "Basic", "price": 29, "featured": false }
  ]
}

// Usage with index tracking:
<!-- monad:loop product in products -->
  <div class="product-{{@index}}">
    {{product.name}} - ${{product.price}}
    Item {{@index1}} of total
  </div>
<!-- monad:endloop -->
