Open Graph Image Generation

Monad can automatically generate beautiful Open Graph images for your pages using the og-gen library.

How it works

Simply add OG metadata to your page frontmatter:

<!-- monad
{
  "og": {
    "title": "Your Page Title",
    "subtitle": "A compelling subtitle",
    "background": "linear-gradient(135deg, #667eea 0%, #764ba2 100%)"
  }
}
-->

Enable OG Generation

In your vite.config.ts:

monad({
  og: {
    enabled: true,
    provider: "og-gen",
    mode: "per-route"
  }
})

Benefits

  • ✅ Automatic generation at build time
  • ✅ Consistent branding across all pages
  • ✅ Better social media engagement
  • ✅ No external services required
