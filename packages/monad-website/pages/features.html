Monad Features

Monad comes packed with features to make building static sites fast, efficient, and enjoyable.

Performance Optimization

Critical CSS Preloading

Automatically extracts and preloads critical CSS for faster initial page renders.

<link rel="preload" href="critical.css" as="style">

Lazy Image Loading

Images below the fold are automatically lazy-loaded with proper loading attributes.

<img loading="lazy" fetchpriority="low">

Image Optimization

Monad automatically optimizes images with WebP/AVIF conversion and responsive sizes.

<picture>
  <source type="image/webp" srcset="image.webp">
  <source type="image/avif" srcset="image.avif">
  <img src="image.jpg" alt="Optimized image" loading="lazy">
</picture>

Content Collections

Manage repeating content with JSON data files. Perfect for testimonials, team members, and more.

// collections.json
{
  "features": [
    { "title": "Fast" },
    { "title": "Simple" }
  ]
}

// In your template:
<!-- monad:loop item in features -->
  <div>{{item.title}}</div>
<!-- monad:endloop -->

Markdown Support

GitHub Flavored Markdown

  • Tables support
  • Task lists
  • Strikethrough text
  • Autolink literals

Smart Typography

  • Smart quotes: "Hello"
  • Em dashes: —
  • Ellipsis: ...
  • Typographic replacements

SEO & Auditing

✓ Build-time SEO Audit

Monad automatically checks your pages for common SEO issues:

  • Missing meta titles or descriptions
  • Duplicate H1 tags
  • Broken internal links
  • Missing canonical URLs
  • Images without alt text

Deploy Anywhere

Built for modern static hosting platforms with automatic integration.

Netlify

Auto-generates _redirects file and supports Netlify Forms.

Vercel

Auto-generates vercel.json with redirect rules.

Cloudflare Pages

Optimized for edge deployment with Workers support.
