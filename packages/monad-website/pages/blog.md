# Our Blog

Welcome to the **Monad** blog! Here we share updates about our static site generator.

## Recent Posts

### Getting Started with Monad

Monad makes it easy to build fast, modern static sites. Here's what you need to know:

- Write pages in HTML or Markdown
- Use partials for reusable components
- Get automatic SEO optimization
- Deploy anywhere with ease

### Why We Built Monad

We wanted a static site generator that was:

1. **Fast** - Built on Vite for lightning-fast development
2. **Simple** - No complex configuration required
3. **Flexible** - Support for HTML, Markdown, and custom templates
4. **Production-ready** - Built-in SEO, performance audits, and more

Check out our [GitHub repository](https://github.com/anthropics/monad) to get started with Monad.
